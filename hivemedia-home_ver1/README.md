# HIVEMEDIA 웹사이트

부산 디지털 마케팅 전문 에이전시 하이브미디어 공식 웹사이트입니다.

## 📁 프로젝트 구조

```
frontend-w-clone/
├── index.html              # 메인 페이지
├── about.html              # 회사 소개
├── services.html           # 서비스 안내
├── portfolio.html          # 포트폴리오
├── contact.html            # 문의하기
├── config/
│   └── content.json        # 콘텐츠 데이터
├── public/
│   └── images/             # 이미지 에셋
├── src/
│   ├── css/
│   │   ├── app.css         # 메인 스타일
│   │   └── pages.css       # 페이지 스타일
│   └── js/
│       ├── app.js          # 메인 로직
│       ├── content-loader.js
│       └── pages.js        # 페이지 기능
└── docs/
    └── STRUCTURE.md        # 구조 문서
```

## 🛠️ 사용 기술

- **HTML5** / **CSS3** / **JavaScript (ES6+)**
- **GSAP 3.x** + **ScrollTrigger** - 애니메이션
- **jQuery 3.x** - DOM 조작

## ✨ 주요 기능

- 커스텀 커서 인터랙션
- 프리로더 애니메이션
- 스크롤 기반 GSAP 애니메이션
- 반응형 모바일 메뉴
- 동적 콘텐츠 로딩 (JSON)
- FAQ 아코디언
- 브레드크럼 네비게이션

## 🎨 SEO 최적화

- Google/Naver 최적화 메타 태그
- Open Graph 태그
- JSON-LD 스키마 (Organization, BreadcrumbList, FAQPage 등)
- 시맨틱 HTML5 마크업

## 🚀 시작하기

1. XAMPP htdocs에 프로젝트 배치
2. `http://localhost/hievemedia_homepage/frontend-w-clone/` 접속
